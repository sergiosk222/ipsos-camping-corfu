# Ipsos Camping Corfu

A modern static landing page for **Ipsos Camping Corfu / Ipsos Pasteria & Camping / Da Marco**.

## What is included

- Responsive one-page website
- Green Mediterranean camping visual style
- Sea, grass, trees and camper van illustration
- No dependency on large photo galleries
- Elegant placeholders for future phone photos
- Call, WhatsApp and Google Maps actions
- SEO title, meta description and keywords
- GitHub Pages friendly static files


## Publish with GitHub Pages

1. Open repository settings.
2. Go to **Pages**.
3. In **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
4. Save.

Your site should become available at:

```text
https://sergiosk222.github.io/ipsos-camping-corfu/
```

GitHub Pages may need several minutes to publish the site.

## Add real photos later

The current page uses visual placeholders because real photos may be limited or low quality. Later, add real photos and replace the gallery cards in `index.html`.

Recommended photos:

- camping entrance
- camper van area
- tent area
- Da Marco outside
- food/pizza/pasta
- path to Ipsos Beach
- greenery/trees/grass
